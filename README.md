# ESP32_BPC_Transmitter
Trasmit BPC time signal with ESP32 and a customized PCB

<img width="1024" height="768" alt="video_logo" src="https://github.com/user-attachments/assets/7ff4ca94-7a73-4c4e-a173-5e7e7dae01c3" />

Hardware:Fine tune the LC parameter to 68.5Khz resonance frequency

Software:Modify the wifi ssid and wifi password,flash the micropython firmware and upload the python code

The transmission distance is approximately two meters, but it will vary depending on your inductor.
